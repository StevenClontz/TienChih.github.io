Section 4.4 Quantifiers

In this section we introduce quantifiers. Sometimes, we want to make statements about something always happening (Everyone who gets hurt should...). Sometimes we want to make a statement about something that will happen but not all the time (Out of all cattle on a ranch, there will be some that...). Sometimes we will want to combine these (For every student in the school, there will be a professor that...). Here, we will discuss how to state such things formally.

Subsection 4.4.1 For All

Consider the statement "Every student at State U likes ice cream." How could we say that formally? Suppose \(x\) is a State U student and \(I(x)\) is the statement “\(x\) likes ice cream.”. Just saying \(I(x)\) is fine for each individual student \(x\text{,}\) but our goal is to say something about all State U students. So we introduce the quantifiers \(\forall\).

The \(\forall\) is pronounced “for all” or “for every”, and it indicates that the statement that follows is true for all values of \(x\text{.}\) So \(\forall x, I(x)\) would denote “For every State U student \(x\text{,}\) \(x\) likes Ice Cream.”. In order for a \(\forall\) statement to be true, it has to be true for every possible value of \(x\text{.}\)

More generally, if \(S(x)\) is a statement about \(x\text{,}\) then

\begin{equation*} \forall x, S(x) \end{equation*}

means that \(S(x)\) is true for every choice of \(x\text{.}\)

Example 4.4.1.

For each of the following sentences, write the formal accompanying statement.

  1. "Every child watches YouTube." Let \(x\) be a child, \(Y(x)\) denote "\(x\) watches Youtube".
  2. "Each problem has a solution." Let \(x\) be a problem, \(S(x)\) denote "\(x\) has a solution".
  3. "All square numbers are non-prime." Let \(x\) be a square number, \(P(x)\) denote "\(x\) is a prime".
  4. "For any integer, if it is prime, then it is either two or odd." Let \(x\) be an integer, \(P(x)\) denote "\(x\) is a prime", \(T(x)\) denote "\(x=2\)", and \(O(x)\) denote "\(x\) is odd".
Solution

Consider these statements:

  1. \(\forall x, Y(x)\text{.}\)
  2. \(\forall x, S(x)\text{.}\)
  3. \(\forall x, \sim P(x)\text{.}\)
  4. \(\forall x, P(x) \to (T(x)\vee O(x))\text{.}\)
Example 4.4.2.

For each of the following statements, write out the sentence accompanying it.

  1. \(\forall x, M(x)\text{,}\) where \(x\) is an accordianist, \(M(x)\) denotes "\(x\) has a trained monkey".
  2. \(\forall x, \sim L(x)\text{.}\) Let \(x\) be a joke, \(L(x)\) denotes "the audience laughed at \(x\)".
  3. \(\forall x, F(x)\to E(x)\) Let \(x\) be an integer, \(F(x)\) denotes "\(x\) is a mutliple of four", and \(E(x)\) denote "\(x\) is even".
  4. \(\forall x, S(x)\wedge I(x)\) Let \(x\) be a linear function, \(S(x)\) denotes "\(x\) has a slope", \(I(x)\) denotes "\(x\) has a \(y\)-intercept".
Solution

Consider these sentences:

  1. "Each accordianist has a trained monkey."
  2. "No one laughed at any of the jokes."
  3. "For each integer, if it's a multiple of four, then it's even."
  4. "Each linear function has a slope and \(y\)-intercept."

Subsection 4.4.2 Exists

The \(\exists\) pronounced “there exists” or “there is” indicates the following statement is true for some values of \(x\text{.}\) It could be true for 1 of them, or a bunch of them, or for all of them. The only thing we know for sure is that there is at least one value of \(x\) for which it is true. So \(\exists x, I(x)\) would mean “There is a State U student who likes ice cream.” or “At least one State U student likes ice cream.” In order for a \(\exists\) statement to be true, it has to be true for at least one, possibly more, choice(s) of \(x\text{.}\)

More generally, if \(S(x)\) is a statement about \(x\text{,}\) then

\begin{equation*} \exists x, S(x) \end{equation*}

means that \(S(x)\) is true for at least one choice of \(x\text{.}\)

Example 4.4.3.

For each of the following sentences, write the formal accompanying statement.

  1. "There is a type of candy which is my favourite." Let \(x\) be a type of candy, \(F(x)\) denote "\(x\) is my favourite".
  2. "There is a movie that I don't like." Let \(x\) be a movie, \(L(x)\) denote "I like \(x\)".
  3. "There is at least one function that is both even and odd." Let \(x\) be a function, \(E(x)\) denote "\(x\) is even", \(O(x)\) denote "\(x\) is odd".
  4. "There is at least one integer both a square number and not even." Let \(x\) be an integer, \(E(x)\) denote "\(x\) is even", and \(S(x)\) denote "\(x\) is square".
Solution

Consider these statements:

  1. \(\exists x, F(x)\text{.}\)
  2. \(\exists x, \sim L(x)\text{.}\)
  3. \(\exists x, E(x)\wedge O(x)\text{.}\) For each \(x\text{,}\) \(x\) is not prime.
  4. \(\forall x, S(x) \wedge \sim E(x)\text{.}\)
Example 4.4.4.

For each of the following statements, write out the sentence accompanying it.

  1. \(\exists x, P(x)\text{,}\) where \(x\) is a student, \(P(x)\) denotes "\(x\) has to present to the class".
  2. \(\exists x, \sim R(x)\text{.}\) Let \(x\) be a city bus, \(Y(x)\) denotes "\(x\) runs late today".
  3. \(\exists x, P(x)\wedge E(x)\) Let \(x\) be a integer, \(P(x)\) denotes "\(x\) is prime", and \(E(x)\) denotes "\(x\) is even".
  4. \(\exists x, E(x)\to Z(x)\) Let \(x\) be a function, \(Z(x)\) denotes "\(x\) has a output zero", \(E(x)\) denotes "\(x\) has an even integer input".
Solution

Consider these sentences:

  1. "There is at least one student who will present to the class today."
  2. "There is at least one city bus who won't run late today."
  3. "There is at least one integer who is prime and even."
  4. "There is at least one function who, whenever it has an even input, it outputs zero."

Subsection 4.4.3 Negating Quantifiers

Let's consider \(\forall x, I(x)\) or “Every State U student likes ice Cream.” If this were not true, what would that mean? It would mean that there is some student at State U who does not like ice cream. In other words: \(\exists x, \sim I(x)\text{.}\)

What would the negation of \(\exists x, I(x)\) be? The statement is saying “There is a student who likes Ice Cream.” What would it mean for this not to be true? It would mean that every single student doesn't like Ice Cream, i.e. \(\forall x, \sim I(x)\text{.}\) So our negation rules are:

\begin{align*} \sim(\forall x, p(x))\amp = \exists x, \sim p(x)\\ \sim(\exists x, p(x))\amp = \forall x, \sim p(x) \end{align*}
Example 4.4.5.

Negate \(\forall x, p(x)\to q(x)\)

Solution

Consider these equivalences:

\begin{align*} \sim(\forall x, p(x)\to q(x))\amp = \exists x, \sim(p(x)\to q(x))\\ \amp = \exists x, \sim(\sim p(x)\vee q(x))\\ \amp = \exists x, p(x)\wedge \sim q(x). \end{align*}
Example 4.4.6.

Negate \(\exists x, a(x)\vee b(x)\)

Solution

Consider these equivalences:

\begin{align*} \sim(\exists x, a(x)\vee b(x))\amp = \forall x, \sim(a(x)\vee b(x))\\ \amp = \forall x, \sim a(x)\wedge \sim b(x). \end{align*}
Example 4.4.7.

Translate the sentence "Every real number is either positive or negative" into a logical statement, negate it, and then translate the negation back to a sentence. Let \(x\) be a real number, \(P(x)\) be "\(x\) is positive", and \(N(x)\) be "\(x\) is negative".

Solution

The statement is "\(\forall x, P(x)\vee N(x)\)". It's negation would be:

\begin{align*} \sim(\forall x, P(x)\vee N(x))\amp = \exists x, \sim (P(x)\vee N(x))\\ \amp = \exists x, \sim P(x)\wedge \sim N(x). \end{align*}

This translates to "There is at least one number which is both not even and not odd".

Example 4.4.8.

Translate the sentence "There is a function who if one evaluates it at zero, the output is negative" into a logical statement, negate it, and then translate the negation back to a sentence. Let \(x\) be a function, \(I(x)\) be "\(x\) is evaluated at 0", and \(N(x)\) be "\(x\) has a negative output".

Solution

The statement is "\(\exists x, I(x)\to N(x)\)". It's negation would be:

\begin{align*} \sim(\exists x, I(x)\to N(x))\amp = \forall x, \sim (I(x)\to N(x))\\ \amp = \forall x, \sim (\sim I(x)\vee N(x))\\ \amp = \forall x, I(x)\wedge \sim N(x). \end{align*}

This translates to "For every function, we evaluate at zero and the output is not negative". (This isn't true, but it's the negation of the original statement).
