Using these circuit diagrams, we can model any logical statement. For example \(p\wedge ((q\wedge r) \vee (\sim q \wedge \sim r))\) would be:

\begin{equation*} \begin{circuitikz} \draw (0,0) to[ospst=$p$] (2,0); \draw (2,0) to (2,1); \draw (2,0) to (2,-1); \draw (2,1) to[ospst=$q$] (3,1); \draw (3,1) to[ospst=$r$] (4,1); \draw (2,-1) to[ospst=$\sim q$] (3,-1); \draw (3,-1) to[ospst=$\sim r$] (4,-1); \draw(4,1) to (4,0); \draw(4,-1) to (4,0); \draw(4,0) to (5,0); \end{circuitikz} \end{equation*}
