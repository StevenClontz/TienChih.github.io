Example 2.1.1.
\begin{align*} 2x+3y\amp =\amp 1\\ x+y\amp =\amp 0 \end{align*}

We note that whatever the solutions to this are, they should be the same as the solutions to:

\begin{align*} x+\frac{3}{2}y\amp =\amp \frac{1}{2}\\ x+y\amp =\amp 0 \end{align*}

We've divided the first equation in half, but we haven't actually changed what points will make that equation true. Then:

\begin{align*} x+\frac{3}{2}y\amp =\amp \frac{1}{2}\\ x+y-(x+\frac{3}{2}y)\amp =\amp 0-\frac{1}{2} \end{align*}

Must also be true, since we can add and subtract the same values from 2 sides of an equation and preserve the solutions. By doing this, we kill off the \(x\) in the second equation and we can solve for \(y\text{:}\)

\begin{align*} x+\frac{3}{2}y\amp =\amp \frac{1}{2}\\ -\frac{1}{2}y\amp =\amp -\frac{1}{2} \end{align*}

So \(y=1\) based on the second line.

\begin{align*} x+\frac{3}{2}y\amp =\amp \frac{1}{2}\\ y\amp =\amp 1 \end{align*}

If we take \(-\frac{3}{2}\) times the second line and add it to the first:

\begin{align*} x+\frac{3}{2}y-\frac{3}{2}(y)\amp =\amp \frac{1}{2}-\frac{3}{2}(1)\\ y\amp =\amp 1 \end{align*}

We obtain:

\begin{align*} x\amp =\amp -1\\ y\amp =\amp 1 \end{align*}

We can verify that if we consider the original system:

\begin{align*} 2x+3y\amp =\amp 1\\ x+y\amp =\amp 0 \end{align*}

then by evaluating at \(x=-1, y=1\text{:}\)

\begin{align*} 2(-1)+3(1)=-2+3\amp =\amp 1\\ -1+1\amp =\amp 0 \end{align*}
