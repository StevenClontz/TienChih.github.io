Also, whenever you see a statement \(\forall p(x)\text{,}\) no matter what \(p(x)\) is, it is fair to deduce \(\exists p(x)\) automatically. If somethings true for EVERY \(x\text{,}\) then it also has to be true for at least 1 \(x\text{.}\) The converse is obviously NOT true. If something is true for a single \(x\text{,}\) it doesn't mean it will be true for more.
