Then, if \(a_n\lt 0\text{,}\) everything is flipped upside down. We can summarize as follows:

  • If \(a_n>0, n\) is even, \(p(x)\to \infty \) as \(x\to \pm \infty\text{.}\)
  • If \(a_n\lt 0, n\) is even, \(p(x)\to -\infty \) as \(x\to \pm \infty\text{.}\)
  • If \(a_n>0, n\) is odd, \(p(x)\to \infty \) as \(x\to \infty\text{,}\) and \(p(x)\to -\infty \) as \(x\to -\infty\text{.}\)
  • If \(a_n\lt 0, n\) is odd, \(p(x)\to -\infty \) as \(x\to \infty\text{,}\) and \(p(x)\to \infty \) as \(x\to -\infty\text{.}\)
