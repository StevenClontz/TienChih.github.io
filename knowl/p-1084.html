If we let \(R\) denote “It rains” and \(B\) denote “I am in a bad mood” then we are given \(P(R)=0.2, P(B)=0.1, P(R\cap B)=0.08\) and may fill out the following venn diagram:

  1. This is \(P(R\cup B)\text{,}\) which we could compute \(P(R\cup B)=0.12+0.2+0.8=0.22\) or \(P(R\cup B)=P(R)+P(B)-P(R\cap B)=0.2+0.1-0.08=0.22\text{.}\)
  2. This is \(P(R\cap B^c)\text{,}\) which we can see is \(P(R\cap B^c)=0.12\text{.}\)
  3. This is \(P(R^c\cup B)\text{,}\) which we can see is \(P(R^c\cup B)=0.78+0.02+0.08=0.88\text{.}\)
