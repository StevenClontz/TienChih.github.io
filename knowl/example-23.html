Example2.2.4

Suppose an event with 300 people ends, and the attendees walk from the venue to two restaurants, Xavier's and Yoneda's. They walk along streets Anderson, Birmingham and Central, which are all one way. When the night ends, there are 75 people and Xavier's and 225 at Yoneda's.

Let \(a,b,c\) denote the number of people walking along Anderson, Birmingham and Central.

  1. Find an expression for the amount of people walking along each street. We first set up our systems of equations. We first note that all 300 people have to walk either along either Anderson or Birmingham, so

    \begin{equation*} a+b=300\text{.} \end{equation*}

    Next, the 75 at Xavier is the however many people walked down Anderson, and take away whoever veered off onto Central so:

    \begin{equation*} a-c=75\text{.} \end{equation*}

    The 225 people at Yoneda is everyone coming across Birmingham and Central so

    \begin{equation*} b+c=225\text{.} \end{equation*}

    Thus the system is:

    \begin{align*} a+b\amp =\amp 300\\ a-c\amp =\amp 75\\ b+c\amp =\amp 225\text{,} \end{align*}

    with associated augmented matrix:

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225 \end{array} \right)\text{.} \end{equation*}

    So the reduced row echelon form of this is:

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

    What this tells us is that there is not a unique solution, as the system associated with this matrix is:

    \begin{align*} a-c\amp =\amp 75\\ b+c\amp =\amp 225\text{,} \end{align*}

    Which is two of the original equations. So, depending on what \(c\) is, this will determine how many people traverse Anderson and Birmingham.

  2. If 50 people crossed central, how many people crossed Anderson and Birmingham? So letting \(c=50\text{,}\) we would have:

    \begin{align*} a-c\amp =\amp 75\\ a-50\amp =\amp 75\\ a\amp =\amp 125\\ b+c\amp =\amp 225\\ b+50\amp =\amp 225\\ b\amp =\amp 175\text{,} \end{align*}

    So 125 people across Anderson and 175 across Birmingham. Another way to do this would be to add an equation \(c=50\) to our system:

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 0 \amp 0 \amp 1 \amp 50 \end{array} \right)\text{.} \end{equation*}

    Which has reduced row echelon form:

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 125\\ 0 \amp 1 \amp 0 \amp 175\\ 0 \amp 0 \amp 1\amp 50\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

    That is \(a=125, b=125, c=50\) which is our solution above.

  3. If 100 people go through Anderson, how many go through Birmingham or Central? We can revisit our linear system from (a):

    \begin{align*} a-c\amp =\amp 75\\ 100-c\amp =\amp 75\\ c\amp =\amp 25\\ b+c\amp =\amp 225\\ b+25\amp =\amp 225\\ b\amp =\amp 200\text{,} \end{align*}

    so 200 across Birmingham and 25 across Central. We can also add the linear equation \(a=100\text{:}\)

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 1 \amp 0 \amp 0 \amp 100 \end{array} \right) \end{equation*}

    with reduced row echelon form:

    \begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 100\\ 0 \amp 1 \amp 0 \amp 200\\ 0 \amp 0 \amp 1\amp 25\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

    So 100 across Anderson, 200 across Birmingham and 25 across Central.
