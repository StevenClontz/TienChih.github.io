So we are solving:

\begin{align*} 8000\amp = 500\frac{(1+i)^{12}-1}{i}\\ 16\amp = \frac{(1+i)^{12}-1}{i} \end{align*}

Then, rather than engage in a nightmarish algebraic exercise, we will simply plot \(y=\frac{(1+x)^{12}-1}{x}\) and see when it reaches the value \(y=12\text{.}\) We can observe that these curves intersect when \(x\approx 0.0509\text{.}\)
