Example 6.1.8.

Consider the domain and range of \(g(x)=\frac{2}{3-x} \text{.}\) In this case, we note that 2 divided by any non-zero number is well-defined. However, we cannot divide two by zero. Thus the domain is \((-\infty, 3)\cup (3,\infty)\text{,}\) every real number other than 3 is a valid input, since \(\frac{2}{3-3}=\frac{2}{0}\) is undefined.

Similarly, there is no value we may divide 2 by to achieve 0, and thus the range is \((-\infty, 0)\cup (0,\infty)\text{,}\) every real value other than zero is a valid output.
