Solution 3.3.6.1.

Let \(x\) be the number of days the Michigan factory is run and \(y\) the number of days the Wisconsin Factory is run. The goal is minimizing the cost \(C(x,y)=960000x+75000y\text{.}\) The inequalities we are subject to are:

\begin{align*} 20x+10y\amp \geq 5000,\ \ \text{Cruisers needed}\\ 40x+30y\amp \geq 12000,\ \ \text{Pontoons Needed}\\ 60x+50y\amp \leq 30000,\ \ \text{Maximum Jet Skis}\\ x\amp \geq 0,\ \ \text{ no negative days }\\ y\amp \geq 0,\ \ \text{ no negative days }\\ x\amp \leq 240,\ \ \text{ cap on days }\\ y\amp \leq 240,\ \ \text{ cap on days } \end{align*}

So sketching the inequalities, we obtain the following feasible region with corners: The profit at each corner is as follows:

\begin{equation*} \begin{array}{c|c|c} x \amp y \amp C(x,y)\\ \hline 130 \amp 240 \amp 960000(130)+75000(240)=142800000\\ 150 \amp 200 \amp 960000(150)+75000(200)=159000000\\ 240 \amp 80 \amp 960000(240)+75000(80)=236400000\\ 240 \amp 240 \amp 960000(240)+75000(240)=248400000\end{array} \end{equation*}

So the cost is minimized when you run the Michigan factory for 130 days and the Wisconsin Factory for 240 days.
