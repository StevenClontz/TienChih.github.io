If 50 people crossed central, how many people crossed Anderson and Birmingham? So letting \(c=50\text{,}\) we would have:

\begin{align*} a-c\amp = 75\\ a-50\amp = 75\\ a\amp = 125\\ b+c\amp = 225\\ b+50\amp = 225\\ b\amp = 175\text{,} \end{align*}

So 125 people across Anderson and 175 across Birmingham. Another way to do this would be to add an equation \(c=50\) to our system:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 0 \amp 0 \amp 1 \amp 50 \end{array} \right)\text{.} \end{equation*}

Which has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 125\\ 0 \amp 1 \amp 0 \amp 175\\ 0 \amp 0 \amp 1\amp 50\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

That is \(a=125, b=125, c=50\) which is our solution above.
