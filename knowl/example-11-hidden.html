Lastly, when we look at:

\begin{align*} x+y\amp =\amp 0\\ x+y\amp =\amp 1\text{.} \end{align*}

We can try to kill the \(x\) in the second line as before:

\begin{align*} x+y\amp =\amp 0\\ x+y-(x+y)\amp =\amp 1-0\text{.} \end{align*}

Where we get:

\begin{align*} x+y\amp =\amp 0\\ 0\amp =\amp 1\text{.} \end{align*}

So all you have to do is find a pair \((x,y)\) so that \(0=1\text{...}\) which is not ever going to happen. So this will never have a solution.
