Item 1.

It helps if we can identify some points on these lines. We will work on them one at a time.

  1. Graphically, we can see that \(S(0)=0\text{,}\) thus \((0,0)\) is on the supply line. This also tells us the \(q\)-intercept, \(b=0\text{.}\) We can also see that when \(p=400, S(400)=1200\text{,}\) thus \((400,1200)\) is also on this line. Thus:

    \begin{equation*} m=\frac{1200-0}{400-0}=3\text{.} \end{equation*}

    So \(S(p)=3p\text{.}\)

  2. Graphically, we can see that \(D(0)=5200\text{,}\) thus \((0,5200)\) is on the demand line. This also tells us the \(q\)-intercept, \(b=5200\text{.}\) We can also see that when \(p=400, D(400)=3200\text{,}\) thus \((400,3200)\) is also on this line. Thus:

    \begin{equation*} m=\frac{3200-5200}{400-0}=\frac{-2000}{400}=-5\text{.} \end{equation*}

    So \(D(p)=-5p+5200\text{.}\)
