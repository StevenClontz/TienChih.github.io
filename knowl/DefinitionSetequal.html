Definition 8.1.9.

We say sets \(A, B\) are equal if each element of \(A\) is also an element of \(B\text{,}\) and each element of \(B\) is also an element of \(A\text{.}\) This is denoted \(A=B\text{.}\) Otherwise \(A\neq B\text{.}\)

Note that \(A=B\text{.}\) is equivalent to sayng \(A\subseteq B, B\subseteq A\text{.}\)
