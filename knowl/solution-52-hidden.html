So we have:

\begin{align*} R\amp = 200\\ P\amp = 2500\\ n\amp = 15 \end{align*}

So we are solving:

\begin{align*} 2500\amp = 200\frac{1-(1+i)^{-15}}{i}\\ 12.5\amp = \frac{1-(1+i)^{-15}}{i} \end{align*}

So by plotting \(y=\frac{1-(1+x)^{-15}}{x}\) and \(y=12.5\text{,}\) we can see what the value of \(i\) should be: \(i\approx 0.0237\text{.}\)

So the effective weekly interest she charged was \(2.37\%\text{.}\)
