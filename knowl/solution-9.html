Solution 3.3.5.1.

Let \(x\) be the number of coach tickets and \(y\) the number first class tickets. The airline is maximizing the profit \(P(x,y)=225x+200y\text{.}\) The inequalities we are subject to are:

\begin{align*} x\amp \geq 40,\ \ \text{minimum coach tickets}\\ y \amp \geq 25,\ \ \text{mineral first class tickets}\\ x+y\amp \leq 150,\ \ \text{maximum capacity}\\ x\amp \geq 0,\ \ \text{ no negative tickets }\\ y\amp \geq 0,\ \ \text{ no negative tickets } \end{align*}

So sketching the inequalities, we obtain the following feasible region with corners: The profit at each corner is as follows:

\begin{equation*} \begin{array}{c|c|c} x \amp y \amp C(x,y)\\ \hline 40 \amp 25 \amp 225(40)+200(25)=14000\\ 40 \amp 110 \amp 225(40)+200(110)=31000\\ 125 \amp 025 \amp 225(125)+200(25)=33125\end{array} \end{equation*}

So the profit is maximized when you sell 125 coach tickstes and 25 first class.
