Section 6.3 Rational Functions

Definition 6.3.1.

A rational function is a function \(r(x)\) which may be written as \(r(x)=\frac{p(x)}{q(x)}\) where \(p(x), q(x)\) are both polynomials, and \(q(x)\neq 0\text{.}\)

Now when we say \(q(x)\neq0\text{,}\) we mean that \(q(x)\) is not the polynomial 0, not that \(q(x)\) could never be 0. As before, we will consider the long and short term behavior of polynomials.

Subsection 6.3.1 Long Term Behavior

For Polynomials, what determined the long term behavior of a polynomial was the leading term. So it makes sense that a rational function will be determined the same way. Given

\begin{equation*} r(x)=\frac{a_nx^n+\cdots a_0x^0}{b_mx^m+\cdots b_0x^0}\text{,} \end{equation*}

we can determine the long term behavior with

\begin{equation*} \frac{a_nx^n}{b_mx^m} \end{equation*}

which simplifies to \(\frac{a_n}{b_m}x^{n-m}\text{.}\) So there are some cases to consider here:

  • If \(n>m\text{,}\) then \(\frac{a_n}{b_m}x^{n-m}\) is just a regular monomial. This doesn't mean that \(r(x)\)'s graph will look like a polynomial's, but it does mean the long term behavior will be the same as that of a polynomial. For example consider \(y=\frac{-x^4+x}{x+1}\text{.}\) We should note: \(\frac{-x^4-x}{x+1}\sim\frac{-x^4}{x} \sim -x^3\text{,}\) so the long term behavior of \(y\) will be the same as \(-x^3\)'s: positive infinity to the left, negative infinity to the right. \url{https://www.desmos.com/calculator/jzxyyrcgep}.

  • If \(n=m\text{,}\) then \(\frac{a_n}{b_m}x^{n-m}=\frac{a_n}{b_m}\) is just a constant. So as the magnitude of \(x\) grows, these values approach \(\frac{a_n}{b_m}\text{.}\) For example consider \(y=\frac{-2x^4+x}{3x^4+1}\text{.}\) We should note: \(\frac{-2x^4-x}{3x^4+1}\sim\frac{-2x^4}{3x^4} \sim -\frac{2}{3}\text{,}\) so the long term behavior of \(y\) will be the same as \(-\frac{2}{3}\)'s: \url{https://www.desmos.com/calculator/qaeq86mivp}.

  • If \(n\lt m\text{,}\) then \(\frac{a_n}{b_m}x^{n-m}=\frac{a_n}{b_m}\frac{1}{x^{m-n}}\text{.}\) As the magnitude of \(x\) grows, the more \(\frac{1}{x^{m-n}}\) will shrink and the entire function converges to 0. For example consider \(y=\frac{-2x^2+x}{3x^4+1}\text{.}\) We should note: \(\frac{-2x^2-x}{3x^4+1}\sim\frac{-2x^2}{3x^4} \sim -\frac{2}{3}\frac{1}{x^2}\text{,}\) so the long term behavior of \(y\) will be converging to 0: \url{https://www.desmos.com/calculator/n0ftbxmcod}.

Subsection 6.3.2 Short term behavior

Here, we once again care about the multiplicity of the roots, that is, if \(r(x)=\frac{p(x)}{q(x)}\text{,}\) what we care about is when \(p(x)\) or \(q(x)\) is 0, and what the multiplicity is.

Well, turns out, 0 divided by anything is 0. So if the numerator is 0, then \(r(x)\) is 0. Moreover, it bounces or crosses in the same way as a polynomial. So if the multiplicity is odd, the function crosses and if it's even, it bounces.

For example, consider \(r_1(x)=\frac{x-1}{x}\text{.}\) It is clearly 0 when \(x=1\text{,}\) and since the multiplicity is odd, we cross \url{https://www.desmos.com/calculator/9graa40kq1}. But if we change that to \(r_2(x)=\frac{(x-1)^2}{x}\text{,}\) the multiplicity is now even so we bounce \url{https://www.desmos.com/calculator/iy9kvoac37}.

It also turns out you can't divide by 0. So at those points, the function is undefined. And as the values in the denominator towards 0, the overall value of the quotient approaches either positive or negative infinity. Just like roots, whether or not we change signs at these asymptotes depends on the multiplicity of the root.

For example, again consider \(r_1(x)=\frac{x-1}{x}\text{.}\) The denominator is 0 when \(x=0\text{,}\) so we have a vertical asymptote there. Since the multiplicity is odd, we change signs there: \url{https://www.desmos.com/calculator/ik9xwhui5i}. If we made that multiplicity even, the function would not change sign at \(x=0\text{,}\) so for \(r_2(x)=\frac{x-1}{x^2}\text{:}\) \url{https://www.desmos.com/calculator/h5famtu30t}
